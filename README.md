# ClaudeSAT

AI-assisted development system designed for structured execution, intelligent planning, and continuous improvement.

## Core Capabilities

* Structured planning (Plan Mode)
* Subagent orchestration
* Autonomous execution
* Built-in verification system
* Continuous learning loop

## Philosophy

* Simplicity first
* No temporary fixes
* Senior-level engineering standards

## Structure

```
tasks/   → task tracking & learning
docs/    → system documentation
src/     → core logic
```

## Purpose

ClaudeSAT acts as the cognitive engine behind larger systems like SAT 2.0, enabling scalable and reliable AI-driven workflows.
